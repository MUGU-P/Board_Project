# 자유게시판 만들기



### 학습내용

* MVC, API
* 회원 관리 - 회원 Domain, Repository 생성
* 회원 관리 테스트 케이스 작성
* 회원 서비스
* 회원 서비스 테스트
* h2 DB
* jpa, jdbc
* 스프링 통합 테스트
* 스프링 데이터 jpa
* AOP(Aspect Oriented Programming)


#### 목표 상세 기능

* 게시글 작성 ⭐
* 게시글 수정 ⭐
* 게시글 조회 ⭐
* 게시글 삭제 ⭐
* 이미지 업로드 
* 작성 리스트
* 카테고리
* 댓글 작성 
========================
* 개발 도구
  * IntelliJ
  * OpenJDK 11
  * springboot 2.5.2
  * GITBASH
  * h2 DB 설치
